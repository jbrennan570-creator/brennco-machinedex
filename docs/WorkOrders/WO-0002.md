# WORK ORDER 0002

## Title

Machine Illustration Integration

Created:
July 3, 2026

Status:
Ready

---

## Objective

Replace the temporary illustration placeholders with the beginning of the official MachineDex illustration system.

This milestone establishes the permanent artwork pipeline for every machine.

It is intentionally limited to artwork integration.

No new application features should be added.

---

## References

docs/Design/StyleGuide.md

Use the existing application produced during WO-0001.

---

## Goal

Machine illustrations should become one of the defining visual elements of MachineDex.

Every machine page and machine card should be capable of displaying a consistent pen-and-ink illustration.

---

## Artwork Standard

Illustrations should be:

- Pen and ink
- Technical
- High contrast
- Minimal shading
- White background
- Consistent framing
- Consistent proportions

The illustration style established for the Bliss 110c becomes the standard for all future machines.

---

## Asset Organization

Artwork should live alongside each machine.

Example:

public/

machines/

MDX-001 - Bliss 110c/

artwork/

machine.png

Future machines should follow the exact same structure.

---

## Data Structure

Update the static machine dataset so each machine can reference an illustration.

Example concept:

illustrationPath

The application should gracefully fall back to the placeholder illustration whenever artwork does not yet exist.

---

## User Interface

Machine cards should display the illustration.

The machine detail panel should display a larger version.

Artwork should occupy a prominent visual position without overwhelming the specifications.

---

## Scope

This work order does NOT include:

- Machine search improvements
- Additional machine data
- Dynamic MASTER.txt loading
- Maintenance records
- New navigation
- Database work

---

## Success Criteria

When the application starts:

The Bliss 110c displays its real pen-and-ink illustration.

Machines without artwork continue using placeholders.

The interface looks noticeably more premium while preserving the WO-0001 layout.

---

Provide:

- Files modified

- Asset locations
