# WORK ORDER 0003

## Title

Machine Browser Completion

Created:
July 4, 2026

Status:
Ready

---

## Objective

Complete the visual identity of the Machine Browser.

The current browser successfully establishes the overall MachineDex interface, but the machine cards do not yet match the visual standard established in the MachineDex Vision image.

This work order completes the first "wow factor" of the application.

---

## References

Read:

docs/Design/StyleGuide.md

The StyleGuide is the visual authority for this project.

Also use the MachineDex Vision reference image located in the Design folder.

---

## Goals

### 1. Complete Machine Artwork

Generate consistent pen-and-ink artwork for every machine currently in the static dataset.

Use the Bliss 110c illustration as the visual standard.

Artwork should be:

- Technical
- Pen and ink
- White background
- High contrast
- Minimal shading
- Consistent framing
- Consistent proportions

Artwork should feel like pages from an engineering field manual.

---

### 2. Redesign Machine Cards

Redesign the machine cards to match the portrait layout shown in the MachineDex Vision image.

The current landscape cards should be replaced.

Each machine card should contain:

• MDX ID

• Machine illustration

• Machine name

• Machine classification

• Operational status

Cards should emphasize the artwork first and the specifications second.

The cards should feel collectible while remaining professional.

---

### 3. Improve Grid Cohesion

The machine browser should become visually unified.

Spacing

Alignment

Typography

Borders

Negative space

should all feel intentional and consistent.

The browser should immediately communicate quality.

---

## Preserve

Do NOT redesign:

- Sidebar
- Overall page layout
- Machine Detail page
- Navigation

These were established in WO-0001 and WO-0002.

Only improve the Machine Browser.

---

## Scope

This work order does NOT include:

- Search improvements
- Database work
- Dynamic machine loading
- Maintenance records
- Customer information
- MASTER.txt parsing
- Responsive redesign

---

## Success Criteria

When MachineDex loads:

Every machine displays artwork.

Machine cards use portrait orientation.

The browser closely resembles the MachineDex Vision concept.

The application feels significantly more premium than Version 0.2.
