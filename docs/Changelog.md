# MachineDex Changelog

---

## Version 0.1.0

Date:
July 3, 2026

### WO-0001 — Visual Identity Foundation

Status:
Completed

Summary

Established the first official MachineDex visual identity.

Highlights

- Replaced the generic prototype with a three-panel layout.
- Introduced the engineering field manual aesthetic.
- Implemented the official navy, cream, brass, and steel color palette.
- Created reusable UI components for navigation, machine cards, and information panels.
- Reserved dedicated illustration space for future pen-and-ink machine artwork.

Result

MachineDex now has a recognizable visual identity and a reusable interface foundation.

## Version 0.2.0

Date:
July 3, 2026

### WO-0002 — Machine Illustration Integration

Status:
Completed

Summary

Integrated the permanent machine illustration system.

Highlights

- Added support for machine illustration assets.
- Connected machine records to illustration paths.
- Replaced temporary illustration placeholders where artwork exists.
- Established the artwork pipeline for all future machines.

Result

MachineDex begins to feel like a true engineering encyclopedia rather than a software prototype.

## Version 0.3.0

Date:
July 3, 2026

### WO-0003 — Machine Browser

Completed the first full iteration of the Machine Browser by introducing portrait-oriented machine cards and consistent pen-and-ink artwork for every machine. The browser now closely reflects the established MachineDex visual identity, creating a significantly more cohesive and premium engineering reference experience.
