# Brennco MachineDex

## Purpose

Brennco MachineDex is an interactive digital encyclopedia of Brennco's manufacturing equipment.

The goal is to create a central reference for every machine in the shop by combining machine specifications, photographs, OEM documentation, maintenance records, tooling information, and shop knowledge into a single application.

---

## Current Status

MachineDex is currently in active development.

Current progress includes:

- Machine inventory completed
- MachineDex ID system established
- Master Cards created for all major production machines
- Interactive React prototype running locally
- GitHub repository established

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
